# Real-Time Weather Data Monitoring System (LMQA/UFRGS)

## Description
This project was developed as part of a research initiative at the Meteorology Research Lab (LMQA/UFRGS). It focuses on real-time monitoring and visualization of weather data collected from multiple stations.

## Technologies
- Python  
- Streamlit  
- SQLite  

## Features
- Real-time data visualization dashboards  
- Monitoring of temperature, humidity, pressure, wind, and rainfall  
- Data storage and management using a relational database  

## Data Processing
- Automated data collection and processing  
- Data validation and consistency checks  
- Handling and visualization of structured data  

## Testing & Validation
- Data validation to ensure accuracy and reliability  
- Identification and correction of inconsistencies in data pipelines  

## How to run
Run the Streamlit application to visualize the data in real time.
